# Практика

Бефорд Матвей
М3О-111БВ-25

## Описание программы

Программа выполняет обработку и сортировку данных о посадках самолетов с использованием **индексной сортировки методом "пузырька"**. Основная особенность реализации — отсутствие перемещения тяжелых структур в памяти; сортировка выполняется над вспомогательным массивом индексов.

### Данные о самолетах содержатся в файле, где:
* Каждая строка представляет собой запись о ЛА;
* Программа выполняет жесткий входной контроль (пропуск некорректных строк, проверка формата времени и номера аэродрома).

### Записи имеют структуру:
`Марка_ЛА` `Бортовой_номер` `Время_посадки` `Аэродром`

Выводится таблица записей, сгруппированная по аэродромам и отсортированная по времени посадки (по возрастанию).

**где:**
* **№** - порядковый номер ЛА в отсортированной таблице.

## Компиляция и запуск

`g++ main.cpp -o main
./main`

## Пример вывода

Входные данные загружаются из текстового файла. Программа автоматически добавляет расширение .txt к введенному имени. Например, при вводе test1 будет открыт файл test1.txt.

## Тестовые файлы
- `test1.txt` основной тестовый набор с корректными данными.
- `test2.txt` все записи принадлежат только одному аэродрому.
- `test3.txt` содержит время посадки 00:00 и 23:59.
- `test4.txt` один борт зарегистрирован под разными марками.
- `test5.txt` длинные названия марок ЛА для теста ровности таблицы.
- `test6.txt` записи для разных аэродромов в одинаковое время.
- `test7.txt` корректные записи, расположенные в обратном хронологическом порядке.
- `test8.txt` корректные данные с лишними пробелами между правильными словами.
- `test9.txt` время с нулями в часах и минутах.
- `test10.txt` большой объем данных без ошибок.
- `test11.txt` содержит записи с некорректными часами.
- `test12.txt` содержит записи с некорректными минутами.
- `test13.txt` содержит буквы и символы вместо цифр в формате времени.
- `test14.txt` лишние слова и комментарии в строке.
- `test15.txt` нехватка параметров в строке.
- `test16.txt` некорректный формат бортового номера.
- `test17.txt` некорректные номера и названия аэродромов.
- `test18.txt` два одинаковых БН совершают посадку в одну и ту же минуту.
- `test19.txt` пустой файл для проверки устойчивости программы.
- `test20.txt` мусор, пустые строки и символы вместе с данными.
